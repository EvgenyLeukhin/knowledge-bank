3.1 - string helpers

String helpers

  • .split() - преобразование строки в массив
  • .join() - преобразование массива в строку
  • .indexOf(), .lastIndexOf() - поиск индекса символа с начала / конца строки
  • .slice(), .substring() - вырезать символы из строки
  • .replace(), .replaceAll() - заменить первый найденный символ / все найденные
  • .repeat() - повтор строки
  • .trim() - удалить пробелы в строке
  • .startsWith(), .endsWith() - проверка на начало, конец строки
  • .includes() - проверка на наличие символов в строке
  • .codePointAt() - код симаола в кодировке UTF-16
  • .localeCompare() - сравнение символов по кодировке UTF-16
