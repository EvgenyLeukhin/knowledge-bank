0 JS - Syntax

JS - последняя триада в веб-разработке наряду с html и css, и являющаяся полноценным языком программирования. JS - раньше использовался в осномном просто для выполнения несложных манипуляций с html (показать сообщение, поменять цвет, удалить отдельный блок).

JS в web сегодня - это отдельный полноценный слой web-приложения, который формирует весь пользовательский интерфейс, посылает запросы и обрабатывает полученные данные с сервера.
