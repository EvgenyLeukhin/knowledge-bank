0.1 - document

  • document - корень DOM
  • document.documentElement - <html>
  • document.head - <head>
  • document.body - <body>
  • document.all - вся коллекция DOM
  • document.forms - все формы
  • document.images - все картинки
  • document.links - все ссылки
  • document.scripts - все скрипты
  • document.stylesheets - все стили
  • document.cookie - все куки
  • document.domain - текущее доменное имя
  • document.location - текущая страница
  • document.title - заголовок страницы
  • document.write('Hello World!') - вставка текста в корень
  • ---
  • document.querySelector - общий селектор
  • document.querySelectorAll - общий селектор для коллекций
  • document.getElementById - ... по id
  • document.getElementByName - ... по атрибуту name
  • document.getElementsByClassName - ... по классу
  • document.getElementsByTagName - ... по названию тэга (для коллекции)
  • ---
  • someElement.classList.add('some-class') - добавить класс
  • someElement.classList.remove('some-class') - удалить класс
  • someElement.classList.toggle('some-class') - туглить класс
  • someElement.classList.contains('some-class') - наличие класса
  • ---
  • someElement.removeAttribute('style') - удаление аттрибута
  • someElement.hasAttribute('style') - наличие аттрибута
  • someElement.getAttribute('style') - получение аттрибута
  • someElement.setAttribute('style', 'color: green;') - назначение аттрибута
  • someElement.title - атрибут title
  • someElement.hidden - атрибут hidden
  • someElement.value - атрибут value у элементов форм
  • someElement.dataset.filter = 'some-value' - атрибут dataset
  • ---
  • someElement.style.color = 'red' - изменить стиль
  • someElement.style.backgroundColor = 'pink' -
  • someElement.style.fontSize = '20px' -
  • ---
  • someElement.textContent - текстовый контент
  • someElement.innerHTML - внутренняя разметка
  • someElement.outerHTML - полная разметка
  • ---
  • .createElement() - создание элемента
  • .appendChild() - вставка потомка в конец
  • .insertBefore() - вставить перед
  • .removeChild() - удаление потомка
  • .replaceChild() - замена потомка
  • .remove() - удаление элемента
  • .nextElementSibling() - следующие соседние элементы
  • .previousElementSibling() - предыдущие соседние элементы
  • .firstElementChild - первый потомок
  • .lastElementChild - последний потомок
  • .getHTML() - получение разметки
  • ---
  • .getClientRects() - отступы от окна, координаты
  • .getBoundingClientRect() - размеры элемента
  • .clientWidth, .clientHeight - размеры элемента
