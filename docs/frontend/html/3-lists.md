## Ненумерованный

```html
<ul type="square">
  <li>Пункт №1</li>
  <li>Пункт №2</li>
  <li type="circle">Пункт №3</li>
  <li type="disc">Пункт №4</li>
</ul>
```

<ul type="square">
  <li>Пункт №1</li>
  <li>Пункт №2</li>
  <li type="circle">Пункт №3</li>
  <li type="disc">Пункт №4</li>
</ul>

---

## Нумерованный

**I** - большие римские, **i** - маленькие римские, **a** - мал. буквы, **А** - большие буквы

```html
<ol start="0" type="I">
  <li>Пункт №1</li>
  <li>Пункт №2</li>
  <li>Пункт №3</li>
</ol>
```

<ol start="0" type="I">
  <li>Пункт №1</li>
  <li>Пункт №2</li>
  <li>Пункт №3</li>
</ol>

## Многоуровневый

```html
<ul>
  <li>Пункт №1</li>
  <li>Пункт №2</li>
  <li>
    Пункт №3
    <ul>
      <li>Пункт №3.1</li>
      <li>Пункт №3.2</li>
      <li>
        Пункт №3.3
        <ol type='a'>
          <li>Пункт №3.3.1</li>
          <li>Пункт №3.3.2</li>
        </ol>
      </li>
    </ul>
  </li>
  <li>Пункт №4</li>
</ul>
```

```html
  <dt>Термин №1</dt>
  <dd>Определение №1</dd>
  <dt>Термин №2</dt>
  <dd>Определение №2</dd>
  <dt>Термин №3</dt>
  <dd>Определение №3</dd>
```

```html
<details>
  <summary>Раскройте список </summary>
  <p>текст №1</p>
  <p>текст №2</p>
  <p>текст №3</p>
  <p>текст №4</p>
  <p>текст №5</p>
</details>
```
