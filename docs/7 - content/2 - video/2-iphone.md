## ProRes

### Что такое ProRes

**ProRes** — это профессиональный формат видео, разработанный Apple, который обеспечивает высокое качество и эффективное сжатие для профессиональной видеосъёмки и постобработки.

### Особенности ProRes

- **Минимальное сжатие** с визуально безпотерянным качеством
- **Отличная производительность** при редактировании
- **Поддержка альфа-канала** (в некоторых вариантах)
- **Широкий диапазон битрейтов** (от 45 до 230 Мбит/с)
- **10-битная глубина цвета** (ProRes 422 HQ и выше)

### Варианты ProRes

1. **ProRes 422 Proxy** — самый низкий битрейт, для предварительного просмотра
2. **ProRes 422 LT** — сбалансированный вариант
3. **ProRes 422** — стандартный вариант для большинства задач
4. **ProRes 422 HQ** — высокое качество, 10-бит
5. **ProRes 4444** — максимальное качество с альфа-каналом (на некоторых моделях)

### Доступность

ProRes доступен на iPhone моделей:

- **iPhone 13 Pro** и новее (для 1080p)
- **iPhone 14 Pro** и новее (для 4K)
- Требуется **минимум 256 ГБ** памяти для записи 4K ProRes

### Как включить ProRes

1. Открыть приложение **Камера**
2. Переключиться в режим **Видео**
3. Нажать на **настройки** (иконка настроек вверху)
4. Выбрать **ProRes** (если доступно)
5. Выбрать разрешение: **1080p** или **4K** (зависит от модели)

### Требования для ProRes

- **Минимум 256 ГБ** памяти на устройстве для 4K ProRes
- **Большой объём свободного места** (1 минута 4K ProRes ≈ 6 ГБ)
- **Быстрая карта памяти** для внешнего накопителя (если используется)

### Преимущества

- **Максимальное качество** видео
- **Отличная производительность** при монтаже
- **Гибкость в цветокоррекции** благодаря 10-битной глубине
- **Стандарт индустрии** для профессионального видео
- **Сохранение деталей** в тенях и светах

### Недостатки

- **Огромные размеры файлов** (≈1 ГБ на минуту для 1080p, ≈6 ГБ на минуту для 4K)
- **Быстро забивает память** устройства
- **Требует мощного оборудования** для обработки
- **Не подходит** для обычной видеосъёмки

### Рекомендации по использованию

- Использовать для **профессиональной видеосъёмки**
- Использовать для **коммерческих проектов**
- Использовать, когда нужна **максимальная гибкость** в постобработке
- **Использовать внешний накопитель** для записи длинных роликов
- **Не использовать** для обычных видео, семейных съёмок

### Обработка ProRes

ProRes файлы можно обрабатывать в:

- **Final Cut Pro** (нативный формат Apple)
- **Adobe Premiere Pro**
- **DaVinci Resolve**
- **Avid Media Composer**
- **iMovie** (базовая поддержка)

### Сравнение размеров файлов

| Формат        | Разрешение  | Размер на 1 минуту |
| ------------- | ----------- | ------------------ |
| HEVC (H.265)  | 4K 30fps    | ~150-200 МБ        |
| ProRes 422    | 1080p 30fps | ~1 ГБ              |
| ProRes 422    | 4K 30fps    | ~6 ГБ              |
| ProRes 422 HQ | 4K 30fps    | ~9 ГБ              |
