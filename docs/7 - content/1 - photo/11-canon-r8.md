## Форматы файлов

- Без HDR: Связка `.CR3` + `.JPEG` (raw-файл и обычный jpeg)
- C HDR: Связка `.CR3` + `.HIF` (raw-файл и обычный hif-формат)

**HIF-формат** - это каноновский, основанный на стандарте `HEIF`, для отображения широкого динамического диапозона (HDR). 10 бит,

## Экспорт фото/видео с Canon R8

### I способ (через карту памяти)

Вытащить SD-карту с камеры --> вставить в картридер --> вставить в компьютер

### II способ (через EOS Utility)

С помощью оф. софта Canaon EOS Utility

### III способ (экспорт на смартфон)

Через мобильное приложение "Camera Connect". Можно подключить камеру к смартфону и передавать фото/видео через Wifi/Bluetooth.

Настройки --> Подключить к смартфону --> Открыть приложение на смартфоне

## Обработка HIF

При съёмке фото с HDR будет сохраняться пара `.CR3` + `.HIF`, причём CR3 хранит HDR-информацию.

Можно редактировать этот raw-файл как обычно, то нужно учитывать, что эта фотография должна выглядеть адекватно в SDR тоже, так как HDR имееть низкую поддержку.

Есть переключатель режима HDR или можно использовать галочку в настройках `Предпросмотр для отображения SDR`.

Если выбрать `Экспорт в DNG`, то изменения не будут применены, а пересохранится оригинальный файл без изменений. Но есть решение ниже.

### Экспорт файла с HDR

Экпорт HDR c изменениями работает только для `.HIF`, для `.CR3` к сожалению не работает, но можно решить эту проблему через плагин:

[LRExportHEIC](https://github.com/milch/LRExportHEIC?tab=readme-ov-file) - плагин для корректного экспорта HDR

`ПКМ` --> `Экспорт` --> `Экспорт...` --> `Диспетчер плагинов` --> Добавить --> Указываем путь до архива

После установки лучше перезагрузить LR.

0. Редатируем `.HIF`.
1. `ПКМ` на `.HIF` --> `Экспорт` --> `Экспорт...`
2. Выбираем плагин `Export HEIC` --> `Insert` (не менять настройки, всё дожно подставиться после нажатия)
3. Кнопка `Экспорт`

Будет сохранён файл с расширением `.jpeg` с поддержкой HDR, но его можно в ручную пересохранить в `.heic`.

Таким образом, можно снимать HDR-фото, делать его последующую обработку и экспортировать готовое HDR-фото.
