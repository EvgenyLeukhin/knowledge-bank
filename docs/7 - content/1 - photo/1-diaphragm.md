Светосила (диафрагма)

"Дырка" объектива. Влияет на резкость изображения (глубина резкости). Можно размыть задний план, эффект БОКЕ (открытая диафрагма), либо включить задний план (закрытая диафрагма - нужен светосильный объектив).

Лучше избегать крайних значений диафрагмы на недорогих объективах (будет негативно влиять на качество фото "эффект дымки, замыленности фона" - но можно использовать для композиции)

Средние значения (4, 5.6, 8.0), как правило, дают большую резкость.

На фиксах, как правило, лучшая светосила и общее качество фото.

- Светосильный - f/2,8 - 4,5
- Сверхсветосильный - до f/2,8

## Что означают числа диафрагмы (f-stop)?

Числа диафрагмы (например, 1.2, 1.8, 2.4, 4.0, 5.6, 8.0) — это **относительное отверстие** объектива, выраженное через формулу:. **f-число = фокусное расстояние / диаметр отверстия диафрагмы**

Например, **f/1.2** означает, что диаметр отверстия равен фокусному расстоянию, делённому на 1.2.

### Важные принципы

1. **Меньше число = больше отверстие = больше света**

   - f/1.2 — очень большое отверстие (максимум света)
   - f/2.4 — меньше отверстие (меньше света)
   - f/8 — маленькое отверстие (мало света)

2. **Стандартная шкала диафрагмы** (каждая следующая ступень пропускает в 2 раза меньше света):

   - f/1.0 → f/1.4 → f/2.0 → f/2.8 → f/4.0 → f/5.6 → f/8.0 → f/11 → f/16 → f/22

3. **Промежуточные значения** (например, 1.2, 1.8, 2.4) — это промежуточные ступени между основными значениями, позволяющие более точную настройку экспозиции.

### Почему именно такие числа?

Числа диафрагмы идут в **геометрической прогрессии** с коэффициентом **√2 (≈1.414)**. Это связано с тем, что:

- Площадь круга пропорциональна квадрату диаметра
- Чтобы уменьшить количество света в 2 раза, нужно уменьшить площадь отверстия в 2 раза
- Для этого диаметр нужно уменьшить в √2 раза

### Практическое значение значений

- **f/1.2–f/1.8** — сверхсветосильные объективы, минимальная глубина резкости, сильное размытие фона
- **f/2.4–f/2.8** — светосильные объективы, хорошее размытие фона
- **f/4–f/5.6** — средние значения, баланс между светом и резкостью
- **f/8–f/11** — оптимальная резкость для большинства объективов
- **f/16–f/22** — максимальная глубина резкости, но возможна дифракция (снижение резкости)
