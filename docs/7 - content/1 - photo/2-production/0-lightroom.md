## Установка

- [Убрать системную блокировку установки](https://appstorrent.ru/200-mistakes.html)
- [Запуск программы от рута](https://appstorrent.ru/65-gatekeeper.html)

1. Запустить установщик `Adobe Lightroom Classic v14.5.1 Adobe Activation Tool.dmg`
2. Подтвердить открытие в `Системные настройки` --> `Конфиденциальность и безопасность`
3. Открываем образ установщика (ПКМ --> Открыть)
4. Установить утилиту `Creative Cloud` --> (`1. Creative Cloud` --> `Adobe Creative Cloud 6.7.0 Universal.pkg`)
5. Установить сам `Lightroom` -> (`2. Install` --> `Install` --> `ПКМ` --> Открыть содержимое пакета --> `Install/Contains/MacOs/Install`)
6. Установить активатор `Adobe Activation Tool v1.2.3` --> Запустить (возможны проблемы с запуском `Adobe Activation Tool`) --> Решаем проблемы с запуском

---

### Решение проблемы запуска Adobe Activation Tool

Не открывается программа `Adobe Activation Tool`

- Запустить от рута: `sudo xattr -r -c '/Applications/Adobe Activation Tool.app'`
- Попробовать запустить скрипт `Fix`

## Начало работы

### Импорт библиотеки

- Нажимаем на `Импорт...`
- Выбираем свою корневую директорию с каталогом фотографий
- Вся библиотека будет отображаться в директории `Папки

---

### Импорт фото

- Подключить камеру по USB
- Библиотека --> Импорт --> Выбираем SD-карту/
- Должно открыться окно импорта
- Выбрать всё --> Импорт

---

### Обновление каталого после загрузки новых фото

`Папки` --> Выбираем директорию, в которой есть новые фото --> ПКМ --> `Синхронизировать папку`

### Настройки

- `Основные` --> Считать файлы JPEG рядом с файлами RAW отдельными фотографиями (если пары не схлопнулись, то может помочь команда `Синхронизировать папку`)
- `Стили` --> Включить редактирование HDR по умолчанию для фотографий HDR

### Горячие клавиши (общие)

- Переключатель аватосинхронизации, радом с кнопкой `Сброс`

- `P` - отметить флажком
- `X` - метка на удаление (перенести в отклонённые)
- `U` - удалить метки
- `T` - панель инструментов
- `I` - информация о файле, мета-данные
- `F` - полно-экранный просмотр
- `Cmd + Shift + F` - полно-экранный просмотр c фильтрами
- `L` - затемнить интерфейс
- `[ ]` - рейтинг
- `D` - перейти в Develop
- `G` - grid, сетка библиотеки
- `Z` - zoom

---

### Горячие клавиши (для удаления)

- `X` - метка на удаление (перенести в отклонённые)
- `Cmd + Delete` - открыть отклонённые

---

### Горячие клавиши (в Develop)

- `Cmd + Shift + C` - копировать настройки
- `Cmd + Shift + V` - вставить настройки (при выделении нескольких кадров)
- `Cmd + Shift + S` - синхронизировать настройки (при выделении нескольких кадров)
- `Cmd + E` - экспорт выделенных файлов
- `\` - просмотр фото до обработки
- `S` - получение пробы

### Инструменты (в Develop)

- `Q` - открыть "Ластик" (режим удаления объектов)
- `W` - открыть "Пипетка" (узнать цвет)
- `R` - открыть "Кадрирование"
- `X` - кадрировать (если нажать 2 раза, то будет вертикальное)
- `K` - открыть "Маски"
