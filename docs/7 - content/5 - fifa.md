## Основа

| Кнопка                | С мячом               | Без мяча             |
| --------------------- | --------------------- | -------------------- |
| **СТИКИ (sticks)**    |                       |                      |
| 🕹️LS                  | Направление движения  | Направление движения |
| 🕹️RS                  | Финты                 | Смена игрока         |
| **КУРКИ (triggers)**  |                       |                      |
| LT                    | Филигранные приёиы    | Активная оборона     |
| RT                    | Ускорение             | Ускорение            |
| **БАМПЕРЫ (bampers)** |                       |                      |
| LB                    | Забегание партнёра    | Смена игрока         |
| RB                    | Доп. опции            | Запрос поддержки     |
| **КНОПКИ (buttons)**  |                       |                      |
| 🟢A                   | Пас                   | Толчок               |
| 🔴B                   | Удар по воротам       | Отбор                |
| 🔵X                   | Навес                 | Подкат               |
| 🟡Y                   | Пас вразрез (наперёд) | Выход вратаря        |

---

## Движение с мячом

- 🕹️LS - направление движения
- 🕹️LS + RT - движение с ускорением
- 🕹️LS (клик) + 🕹️RS (клик) - зафиксировать движение игрока
- LB - отправить партнёра в забегание
- RB - запросить поддерку
- RB + RB - запросить поддерку 2-х игроков
- RT + R - проброс на ход
- LT - укрывание мяча
- L + LB - дриблинг в сторону (базовый дриблинг с мячом)
- R - особые приёмы

---

## Пас 🟢A

- 🟢A - простой пас
- 🟢A + 🟢A - с подбросом
- LB + 🟢A - стеночка (отдающий делает забег вперед)
- RB + 🟢A - мощный пас низом
- LT + 🟢A - филигранный пас низом
- RB + LT + 🟢A - филигранный пас низом с подкруткой

---

## Пас вразрез 🟡Y

- 🟡Y - пас вразрез
- 🟡Y + 🟡Y - с подбросом
- LB + 🟡Y - навес врезрез верхом
- RB + 🟡Y - управляемая передача вразрез
- LT + 🟡Y - филигранная передача вразрез с подкруткой
- RB + LT + 🟡Y - управляемая филигранная передача вразрез с подкруткой
- LB + RB + 🟡Y - мощный пас вразрез

---

## Удар 🔴B

- 🔴B - простой удар
- 🔴B + 🔴B - замах и тайминг удара
- RB + 🔴B - прицельный удар
- LB + 🔴B - удар-парашут
- LB + RB + 🔴B - пушечный удар
- LT + 🔴B - удар шведой
- LT + RT + 🔴B - принудительный удар головой с навеса

---

## Навес 🔵X

- 🔵X - простой навес
- 🔵X + 🔵X - прострел
- LB + RB + 🔵X + 🔵X - мощный прострел
- LB + 🔵X - навес-парашут
- RB + 🔵X - управляемый стиком навес
- LT + 🔵X - филигранный навес

---

## Защита

- 🔴B - простой отбор
- 🔴B + RB - жёсткий отбор
- 🔵X - подкат
- 🔵X + RB - жёсткий подкат
- 🟢A - толчок
- 🟢A + RB - фол последней надежды
- LT - активная оборона
- RB - сдерживажние вторым игроком при защите

---

## Вратарь

- 🟡Y - выход из ворот при атаке 1 на 1
- 🟡Y + 🟡Y - выход из ворот в центр поля
- 🟢A - вбрасывание рукой
- RB + 🟢A - мощное вбрасывание рукой
- 🔵X - вбрасывание ногой
- RB + 🔵X - мощное вбрасывание руками
- RT - сбросить мяч себе под ноги
- 🕹️LS - перемещение при пенальти
- 🕹️RS - прыжок за мячом при пенальти
- 🟢A, 🔴B, 🔵X, 🟡Y - ложные махи при пенальти

---

## Финты

- ←R + R→ + LB - "радуга"
- R кручение от 3/9 часов до 6-ти по длинной дуге - финт Зидана (Марсельская рулетка)
