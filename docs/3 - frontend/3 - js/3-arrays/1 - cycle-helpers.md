---

- .forEach() — перебор без формирования нового массива (часто путают с .map()).
- .flatMap() — «map + flat на 1 уровень» в одном проходе.
- .findLast() / .findLastIndex() — поиск с конца (удобно, когда нужен последний матч).
- .toSorted() / .toReversed() / .toSpliced() — современные не-мутирующие аналоги .sort() / .reverse() / .splice() (возвращают новый массив).
- Итераторы: .entries() / .keys() / .values() — полезно для for..of, когда нужны индексы/пары.
